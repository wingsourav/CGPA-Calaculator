# 🎓 CGPA Calculator

A modern, responsive **CGPA Calculator** web application that helps students accurately calculate their **CGPA** using either **Subject-wise** or **Semester-wise** calculation methods. The application includes a complete authentication system with account creation, login, password recovery, and password change functionality. It automatically saves user data locally, supports both **Light** and **Dark** themes, and provides a clean, intuitive user interface.

Built entirely with **HTML**, **CSS**, and **Vanilla JavaScript**, this project demonstrates frontend development, client-side authentication, local storage management, and responsive web design.

---

## ✨ Features

- 🎓 Subject-wise CGPA Calculator
- 📊 Semester-wise CGPA Calculator
- 👤 User Registration & Login
- 🔒 Strong Password Validation
- 🔄 Change Password
- 🔑 Forgot Password Recovery
- 💾 Automatic Data Saving using Local Storage
- 🌙 Light & Dark Theme Support
- 📱 Fully Responsive Design
- ⚡ Built with HTML, CSS & Vanilla JavaScript (No Frameworks)
