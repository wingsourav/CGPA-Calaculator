# CGPA-Calaculator
CGPA Calculator is a responsive web application that helps students accurately calculate their CGPA through two calculation modes: subject-wise and semester-wise. The application includes a complete authentication system with account creation, login, password recovery, and password change functionality. It automatically saves user data locally, supports dark and light themes, and provides a clean, user-friendly interface for managing academic performance. Built entirely with HTML, CSS, and vanilla JavaScript, the project demonstrates frontend development, client-side authentication, local storage management, and responsive UI design.

Features
🎓 Subject-wise CGPA Calculator
📊 Semester-wise CGPA Calculator
👤 User Registration & Login
🔒 Strong Password Validation
🔄 Change & Recover Password
💾 Auto-save using Local Storage
🌙 Dark & Light Mode
📱 Fully Responsive Design
⚡ Built with HTML, CSS & JavaScript (No Frameworks)
